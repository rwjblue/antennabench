# AntennaBench Examples

Examples are optional local integrations, not built-in device support. Review
an example before attaching it to a controller profile: it runs with the same
operating-system authority as AntennaBench.

## Rig Control

- [Elecraft K4 through QK4](rig-control/elecraft-k4/) switches and verifies
  KAT4 ANT1/ANT2 for transmit and main-receiver use without a shell.
