# Coverage map mockups for issue #203

Temporary design-exploration artifacts for
[#203](https://github.com/rwjblue/antennabench/issues/203) (three-state
coverage map). Not authoritative documentation — the issue body and its
approved design note govern.

`coverage-map-mockups.html` is fully self-contained (open it directly in a
browser; no network, no scripts). It demonstrates, with real data from
session `n1rwj-20260719T022454Z` (20m WSPR-2, 2026-07-19 02:28/02:30 UTC
cycles, census of all active wspr.live reporters):

- the equirectangular maidenhead-square view and the azimuthal-equidistant
  per-reporter view, combined into one figure block behind a **script-free
  CSS toggle** (hidden radio inputs + `:checked` sibling selectors; print
  CSS forces the grid view) — proof the toggle works under a no-script CSP;
- the **compact polar-cell view**: 8 bearing sectors × 4 √-scaled distance
  rings (edges 1 / 3 / 8 / 20 Mm), 32 three-state cells per antenna;
- the three-state legend, per-cell/per-dot `<title>` tooltips, the numbers
  table used as the accessible fallback, and light/dark theming via CSS
  custom properties.

Screenshots (light grid view, light polar view, compact polar cells, dark
grid view) are referenced from the issue for sessions without a browser.

Provenance details (station FN41FR; land outline from public-domain Natural
Earth 110m, embedded as inline SVG paths) are in the page footer. The page
was generated by a throwaway script; regenerating it is not required — edit
or rebuild freely if the design note needs different variants.
