04 Run plan

Choose the intended antenna order. Actual WSPR cycle times are set during the run after each antenna is ready.

Complete rounds One round tests every scheduled antenna in each direction selected by the mode.

4 rounds · 16 planned WSPR cycles · about 32 minutes of required cycle time · then a 5-minute WSPR.live ingestion grace.

Antenna switching, confirmation, eligible-cycle boundary waits, the final WSPR.live request, and recovery from failure can add wall-clock time. When automatic WSPR.live collection is enabled, its five-minute grace is not a guarantee that every report has arrived.

05 Antenna switching assistant

Manual switching remains the default. Turn this on to let AntennaBench run a controller program on this computer before a planned cycle. Enabling it gives the new session temporary permission to use the reviewed profile.

Antenna targets

Enter the controller value substituted for {target}, such as a relay number, port name, or switch position.

Keep this checked for manual review. If turned off, both the switch and verification commands must succeed before AntennaBench schedules the next eligible WSPR cycle.

Before sharing a session bundle, review controller command details and output. They may contain local paths, usernames, network addresses, or credentials.

Creating the session also saves the reviewed profile on this computer. This temporary controller permission is revoked when the app closes or the run is paused.

06 WSPR Spots

Delayed public TX and RX WSPR spots are collected automatically from WSPR.live, a volunteer WSPRnet mirror. Enable WSJT-X Upload spots and keep WSJT-X online. AntennaBench collects returned rows for configured request windows on a best-effort basis; the upstream mirror does not provide an independent completeness guarantee.
