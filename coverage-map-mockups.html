COMPACT VIEW

Each cell keeps the honest three states — a quiet cell is visibly "no active receivers", never "not reached". FOR THE DESIGN NOTE

Decisions this page proposes

Ship both projections behind the script-free toggle (grid squares default and print view; bearing & distance as the second tab).

Compact reports get the polar-cell figure plus the table — same polar geometry as the full view, aggregated to 32 cells.

One color story with the directionality figures. The companion "directionality charts" proposal uses a blue↔orange diverging scale for signed antenna deltas. Coverage states must not reuse that pair with a different meaning in the same report. Proposed rule to settle in the design note: entity colors mark "heard" per antenna panel, a hatched treatment marks "active, not heard" (matching the report's existing no-data hatching), and the diverging pair stays reserved for delta figures.

Shared geographic foundation first. Station lat/lon from the grid locator, the azimuthal-equidistant projection, √-scaled ring geometry, and the embedded ~30–60 KB simplified coastline (Natural Earth, public domain, with an attribution entry) should land as one helper that this figure and the directionality figures both consume. Everything renders offline; wspr.live is only contacted during live capture.
